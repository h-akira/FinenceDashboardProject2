# API エンドポイント一覧（暫定）

> 本ドキュメントは暫定版である。承認後、OpenAPI 定義（`openapi_main.yaml`）で置き換える。

## メイン API

ベースパス: `/api/v1/main`

### ユーザー

| メソッド | パス | 認証 | 概要 | 対応 US |
|---------|------|------|------|--------|
| GET | `/users/me` | 要 | ログインユーザーの情報取得 | US-2.1 |
| DELETE | `/users/me` | 要 | アカウント削除（パスワード再検証） | US-2.2 |

### 金融データ

| メソッド | パス | 認証 | 概要 | 対応 US |
|---------|------|------|------|--------|
| GET | `/finance/interest-rate` | 要 | 政策金利・長期金利データの取得 | US-6.1 |
